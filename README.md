# Velir AI – Digital Farmer Officer

Velir AI aims to provide farmers with an accessible digital advisory system that simplifies agricultural decision-making through technology.

Velir AI is a digital agricultural advisory prototype designed to assist farmers with crop guidance, alerts, and farming insights through a simple interface.

This prototype demonstrates how digital systems can support farmers by providing structured access to agricultural information and advisory support.

Developed as part of a prototype submission.

## Problem Statement

Farmers often face difficulties accessing timely and reliable agricultural information. Important advisories related to crops, pests, weather conditions, and best farming practices may not always reach them efficiently.

This project proposes a prototype system that acts as a digital agricultural officer to assist farmers through a simple interactive interface.

---

## Proposed Solution

The system demonstrates how digital platforms can assist farmers by providing accessible agricultural advisory support through an interactive interface.

Farmers can use the platform to:

- Ask agriculture-related questions  
- Upload crop images for analysis  
- View alerts and advisory information  
- Access a simplified farming dashboard  

The goal is to show how cloud-based systems can assist in delivering agricultural knowledge and information.

---

## Features

The Velir AI prototype includes the following core capabilities:

- Farmer dashboard interface  
- Query submission interface for agricultural questions  
- Image upload interface for crop-related issues  
- Display of advisory responses  
- Basic alert and information display  

These features illustrate how farmers could interact with a digital advisory system.

## Process Flow

1. Farmer accesses the web interface  
2. The farmer submits a question or uploads a crop image  
3. The request is processed by the backend service  
4. The system retrieves relevant responses or advisory information  
5. The response is displayed on the farmer dashboard  

This flow illustrates the basic interaction between the farmer and the advisory system.
